# 暴力DFS

Q : 暴力怎么写？ \
A : dfs深搜

Q : 瓶颈在哪里? \
A : 重复计算 O(2 ^ n) 时间复杂度大

Q : 正解消掉了那层复杂度？ \
A : 通过记忆化，DP等手段储存中间数据

Q : 下次看到什么特征要想到这个问题？ \
A : 找DEPTH,想递归，再优化

# DP

我们令 f(i,j) 为球传到 i 时传球次数为 j 的方案数。

显然根据题意，我们需要找到的是 f(1,m)

我们从 1 开始，逐次计算每个 f(i,j) 的值。每次计算，它的值就是 f (i−1,j−1)
​
和 f (i+1,j−1) 的和。

球可以从 i−1 号同学传到 i 号同学，也可以从 i+1 号同学传到 i 号同学。

最终输出 f(1,m)
