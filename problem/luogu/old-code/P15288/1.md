## 题目解析

思考的过程

### 题目在问什么（翻译成简单话）

有多少个k(1,2,3 ... ),使得: (存在一种排列 -> 任意连续k个连续的数字的和一样)  

最基本的思想: 暴力枚举


### 猜想

1. $a_i$很大,时间和$a_i$无关
2. 和具体的n个数字有关,只要知道了 n 个数字是什么,就可以得到 所有的k

> ! 一定存在一种性质: 且是数学上的性质,可以通过计算得到答案

### 结论

```
假设 k =3 成立
a0 a1 a2 a3 a4 a5 ... an

a0 + a1 + a2 = a1 + a2 + a3 -> a0 = a3
同理 -> a2 = a4, a3 = a5

=> 结论: 当k=3成立的时候 ai = a_{ (i+k) mod n} 

意思是：**每隔 (k=3) 个位置，数必须一样**。

也就是说，环上的位置会被分成若干组，每组里的位置必须放同一个值。
(看数据 )
p -> q
```

此时就要思考了 `q ->p` 怎么才能上让=3成立呢? -> 数字可以把k=3 的长度进行分组.

如何得到公式: 所有的` k | cnt_num` 都成立 


如何快速的求出所有的k?

如何k成立,则 2k 也成立!

那怎么求出最小的k呢?

```
mod n 表示走圈

a_i = a_{(i+ x*k ) mod n} i 位置每走 k步 都一样
a_i = a_{ (i + y*n) mod n} i 位置走一圈 都一样

a_ = a_{i + x*k + y*n ) mod n} 现走 xk ,再走 yn 都一样

根据裴蜀定理 存在 一对x,y (x *k + y *n = gcd(k,n))  

若存在k的循环节,则必然存在 gcd(n,k)的循环节
```

那怎么求出最小的k呢? 又是 p -> q , q->p 

存在k, k | cnt_num, k | cnt _num2

k: group 组长,循环节
cnt_of_group: 组的数量 k * cnt_of_group = n  当k最小的时候,cnt_of_group 一定是最大的

```
cnt_of_group | cnt_num_1
cnt_of_group | cnt_num_2
cnt_of_group | cnt_num_3
...
```

那么最小的k,就是对应最大的cnt_of_group, 那么就是: cnt_of_group 所有元素的最大公约数



最小的k, ->  k 

方案二:  `k`是循环长度 -> gcd(n,k) 也就循环长度.
