# 题意

给定一个 $n \times n$ 的矩阵 $a$。小可可从 $(1, 1)$ 出发走到 $(n, n)$，每次只能向下或向右移动。初始时身上携带一个变量 $v = a_{1,1}$，每走到一个格子 $(i, j)$ 时，$v$ 会变为 $\gcd(v, a_{i,j})$。求到达终点时 $v$ 的最大可能值。

# 思路

## How ?

本题要求在矩阵中寻找一条路径，使得依次求 $\gcd$ 后的结果最大。很容易想到 DFS 。

如果我们仅将坐标 $(i, j)$ 作为搜索状态，是无法进行正确记忆化的。这是因为即使到达同一个格子，如果之前走过的路径不同，当前累积的变量 $v$ 也可能不同。 

由于 $\gcd$ 运算的性质，较大的 $v$ 在后续不一定会得到更大的结果（例如遇到 $9$ 时，$v=3$ 比 $v=4$ 能得到更优的 $\gcd$），因此**所有不同的 $v$ 值都必须保留并继续向后搜索**。 

由此，我们需要定义一个记忆化状态：当前位于第 $i$ 行、第 $j$ 列，且当前变量值为 $v$。只要到达同一个格子时 $v$ 的值曾经出现过，就不需要再向下搜索了。

## Optimize :

因为初始值 $v = a_{1,1}$，且每次更新操作为 $v \gets \gcd(v, a_{i,j})$，所以**任何时刻 $v$ 的值都必定是 $a_{1,1}$ 的约数**。

我们可以使用 `set` 来进行状态去重。定义 `set<int> vis[i][j]`，当搜索到时：
1. 检查 $v$ 是否已经在 `vis[i][j]` 中出现过。如果是，说明该状态已经搜索过，直接 `return`。
2. 如果没有出现过，将 $v$ 插入 `vis[i][j]`，并继续向下和向右搜索。

这题一眼想到dp，反正记忆化和dp差不多，当然是写记忆化啦！

# Code

```cpp
#include <bits/stdc++.h>
using namespace std;
const int maxn = 1e6 + 5;
typedef long long ll;

int n, V;
int a[1005][1005]; // 地图
// 普通gcd
int gcd(int a, int b)
{
    return (b == 0) ? a : gcd(b, a % b);
}

// 检查有没有超出边界
bool check(int nowi, int nowj)
{
    if (nowi < 1 || nowi > n)
        return 0;
    if (nowj < 1 || nowj > n)
        return 0;
    return 1;
}

set<int> vis[1005][1005];
int maxans = -1;
void dfs(int nowi, int nowj, int nowv)
{
    if (!check(nowi, nowj))
        return;
    // 记忆化核心 根据不同路径加入set
    if (vis[nowi][nowj].count(nowv))
        return;
    vis[nowi][nowj].insert(nowv);
    // 边界处理
    if (nowi == n && nowj == n)
    {
        maxans = max(maxans, nowv);
        return;
    }

    dfs(nowi + 1, nowj, gcd(nowv, a[nowi + 1][nowj]));
    dfs(nowi, nowj + 1, gcd(nowv, a[nowi][nowj + 1]));
}

signed main(int argc, const char **argv)
{
    ios::sync_with_stdio(0);
    cin.tie(0), cout.tie(0);
    cin >> n >> V;
    for (int i = 1; i <= n; i++)
    {
        for (int j = 1; j <= n; j++)
        {
            cin >> a[i][j];
        }
    }

    dfs(1, 1, a[1][1]);

    cout << maxans << endl;
    // END
    return 0;
}
```
