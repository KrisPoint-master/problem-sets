
证明我写这个代码的证明性： 

1. 倒过来存图
2. 从大到小开始从某个点u（如果u没有被vis过）进行dfs遍历，这时所有访问过的点的能访问过的最大值$f(x) = u$


证明：

1. 在有向图G中如果存在一条从u到v的路径$u \to v$,那么在G的反向图$G‘$中必然存一个路径$v \to u$


> 证明: 数据归纳法
> 设路径$u \to v$,v 的前一点是$k_1$,那么在$G'$中必然存在一条反过来的边$(v,k_1)$,
> 这个问题就变成了求$u \to k_1$与在$G'$中$k_1 \to u$是否成立

2. 根据证明1,得到,$G$如何可以从u到达点v,那么在$G'$中必然可以从v到u(一种二元关系)


3. 综上,可以想到在$G'$中从$Max$点开始遍历,能到的点的集合的点为$A(Max)$,那么在G中$A(Max)$一点可以达到$Max$点,又因为Max是最大点,可以想到$A(Max)$中的点的答案必然是$Max$(可以使用反证法证明)

4. 如果x在Max访问时被访问到,那么再从x访问开始遍历没有意义

上面这句话意思是$A(x) \subset A(Max)$,也就是x访问过到点的集合是Max能访问到的点的集合的子集. 


```

u --> x ---> v
```

因为Max能访问到x,所以存在一条路$u \to x$,又因为x能访问在v,所以同样存在一条路径$x \to v$,把这条路径拼接在一起,得到一条路径$u \to v$

所以$\forall x \in A(x) \to x \in A(max)$


所以$A(x)$集合里点的答案都是Max
