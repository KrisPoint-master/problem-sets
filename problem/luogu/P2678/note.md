# 跳石头 笔记

## 题意

长度为 L 的线段上有 N 块岩石（位置 D_i），最多移走 M 块，求最大化选手跳跃过程中的**最短跳跃距离**。

---

## 二分答案框架

问题具有单调性：如果最短跳跃距离 x 是可行的，那么任意更小的距离也是可行的。
因此我们二分最短跳跃距离，用 check(x) 判断"能否在移走 ≤M 块岩石的前提下，使得所有跳跃距离 ≥ x"。

---

## 二分边界详解

以 `bsfind_byP1824.cpp` 为准：

```cpp
int bs_find(int l, int r) // 搜索区间 [l, r)  左闭右开
{
    while (l < r)          // 区间非空
    {
        int m = (l + r) >> 1;  // 下取整中点
        if (check(m))          // m 可行 → 答案至少为 m，尝试更大
            l = m + 1;         // 左边界右移，排除 m
        else
            r = m;             // m 不可行 → 答案在左边，右边界收缩到 m
    }
    return l - 1;  // 循环结束时 l == r, 且 l 是第一个不可行的值
}
```

### 关键细节

| 要素 | 取值 | 理由 |
|---|---|---|
| 初始 l | `0` | 最短距离至少为 0 |
| 初始 r | `L + 1` | 最短距离最大不可能超过 L；设为 L+1 使得 **L 也在搜索范围内**（因为是开区间 `[l, r)`） |
| Mid 方向 | 下取整 `(l+r)>>1` | 与 `l = m+1, r = m` 的更新方式配套 |
| Check(m) = true | `l = m + 1` | m 可行，答案 ≥ m，尝试更大的值 |
| Check(m) = false | `r = m` | m 不可行，排除 m 及更大值 |
| 返回值 | `l - 1` | 循环结束时 l 指向第一个不可行的值，所以最后一个可行值是 `l - 1` |

### 为什么初始 r = L + 1

因为区间是 `[l, r)` 左闭右开，如果 r = L，最多只能搜到 L-1，L 被排除在外。
而最短跳跃距离理论上可以取到 L（只有起点和终点，中间全移走），所以右端点必须至少为 L+1。

### 为什么返回 l - 1

观察循环不变式：`[0, l)` 内全是可行值，`[r, +∞)` 内全是不可行值。
循环结束 l == r，此时 l 是第一个不可行的值，因此 `l - 1` 是最后一个可行的值，即答案。

## Check 函数详解

```cpp
bool check(int x)
{
    int tot = 0;     // 移除的岩石数量
    int now = 0;     // 上一块保留的岩石的索引（0 表示起点）
    for (int i = 1; i <= n; i++)
    {
        if (a[i] - a[now] < x)   // 距离 < x → 必须移除
            tot++;
        else                     // 距离 ≥ x → 保留
            now = i;
    }
    if (L - a[now] < x)          // 最后一段到终点的距离
        tot++;
    return tot <= m;             // 移除数量 ≤ 限额
}
```

### 思路

贪心地从起点开始，依次检查每块岩石：
- 如果当前岩石与上一块保留的岩石距离 **< x**，说明这一跳太短，必须移走这块岩石（`tot++`），**不更新 now**，下一块岩石继续和 now 比。
- 如果距离 **≥ x**，说明这一跳合格，保留这块岩石（`now = i`）。

遍历完所有岩石后，还要检查最后一段（最后保留的岩石到终点 L）是否 ≥ x，若 < x 则需要移除（实际意味着方案不可行，但通过 tot 计数来体现）。

### 贪心正确性

因为要让"所有跳跃距离 ≥ x"，对于每块岩石，**能留就留**是最优的——早保留意味着后面的岩石有更大的空间。如果一块岩石与上一块的距离已经 ≥ x，就没必要移走它。

---

## 总结

- **二分答案**：最小值最大化问题，标准解法是对答案二分。
- **边界要点**：左闭右开区间 `[l, r)`，配合 `l = m+1, r = m` 和 `return l-1`。
- **右端点初始值**必须足够大（L+1），确保答案在搜索范围内。
- **Check**：贪心扫描，统计不得不移除的岩石数，与 M 比较。
